## 1. User Accounts & Roles

The SURM Student Portal supports distinct user roles, each tailored to specific responsibilities within the Madrasah.

### User Roles & Capabilities

| Role | Description | Key Capabilities |
| :--- | :--- | :--- |
| **Student** | For secondary school students (Sec 1-4). | ‚Ä¢ View upcoming assignments & due dates<br>‚Ä¢ Submit homework (files, text)<br>‚Ä¢ View grades & teacher feedback<br>‚Ä¢ Access learning materials (PDFs, Videos)<br>‚Ä¢ Apply for leave / Submit MCs |
| **Teacher** | For all teaching staff. Can be specialized (Form, Tahfiz, Subject). | ‚Ä¢ Create & manage assignments<br>‚Ä¢ Grade submissions & provide feedback<br>‚Ä¢ Upload learning materials<br>‚Ä¢ View student lists & profiles<br>‚Ä¢ Approve/Reject leave applications (Form Teachers) |
| **Principal** | For school leadership. | ‚Ä¢ **All Teacher capabilities**<br>‚Ä¢ View school-wide statistics<br>‚Ä¢ Monitor teacher performance<br>‚Ä¢ Oversee all classes and levels |
| **Admin** | For system administrators. | ‚Ä¢ Manage user accounts (Create/Edit/Delete)<br>‚Ä¢ System configuration<br>‚Ä¢ Technical troubleshooting |

### Account Management

*   **Registration:** Accounts are currently created by the System Administrator.
*   **Required Fields:** Full Name, Email Address, Role, Level (for Students), Class/Halaqah assignment.
*   **Password Reset:** Please contact the Administrator if you forget your password.
